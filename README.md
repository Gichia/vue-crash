# Vue.js Crash Course
> This is a todo list app created with the vue.js framework.

# Getting Started
```bash
# Clone this repo
$ https://github.com/Gichia/vue-crash.git

# Navigate to the folder
$ cd vue-crash

# Install dependencies
$ npm install

# Start the server
$ npm run serve
```
